What is Routing in Angular?
  • Routing allows navigation between different components.
  • Defined in app-routing.module.ts with path-component mappings.
  • Supports parameters, nested routes, and lazy loading.

Basic Routing Example

Use <a routerLink="/home">Home</a> to navigate.

Router Setup:

const routes: Routes = [
  { path: 'home', component: HomeComponent }
];
Route Parameters

Use routerLink="/user/42" and read with ActivatedRoute.

this.route.snapshot.paramMap.get('id');
Nested Routing

Child routes are defined under a parent route:

const routes: Routes = [
  {
    path: 'dashboard',
    component: DashboardComponent,
    children: [
      { path: 'stats', component: StatsComponent }
    ]
  }
];
Wildcard & Redirects

Redirect unknown routes or define default path:

const routes: Routes = [
  { path: '', redirectTo: '/home', pathMatch: 'full' },
  { path: '**', component: PageNotFoundComponent }
];
