What are Signals?

Signals in Angular are a reactive primitive introduced to manage data changes efficiently. They help track dependencies and auto-update views when values change.

  • signal(): Creates a signal with an initial value.
  • computed(): Defines a derived value based on one or more signals.
  • effect(): Reacts to signal changes and runs side effects.
