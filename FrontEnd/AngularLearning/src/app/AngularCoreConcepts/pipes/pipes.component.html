Introduction to Pipes in Angular

Pipes in Angular help transform data in the template before display.

  • Date Pipe: Formats date values.
  • UpperCase / LowerCase Pipe: Changes the case of text.
  • Currency Pipe: Formats numeric values as currency.
  • Custom Pipe: Your own pipe logic to transform data.

Date Pipe Example

Today is: {{ today | date : 'fullDate' }}

Explanation: {{ '{{ today | date:"fullDate" }}' }} formats the date in a readable form.

UpperCase Pipe Example

Transformed Text: {{ message | uppercase }}

Explanation: {{ '{{ message | uppercase }}' }} transforms the string to uppercase.

Currency Pipe Example

Product Price: {{ price | currency : 'INR' }}

Explanation: {{ '{{ price | currency:"INR" }}' }} formats the number as Indian currency.

Custom Pipe Example

Transformed: {{ 'angular rocks!' | capitalizeWords }}

Explanation: {{ '{{ "angular rocks!" | capitalizeWords }}' }} applies a user-defined transformation to the string.
