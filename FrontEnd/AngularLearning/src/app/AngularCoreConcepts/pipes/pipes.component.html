Pipes in Angular help transform data in the template.

  • Date Pipe: Formats date values.
  • UpperCase/LowerCase Pipe: Changes text case.
  • Currency Pipe: Formats currency values.
  • Custom Pipe: Your own logic to transform data.
  • Date Pipe
  • UpperCase Pipe

Date Pipe Example

Today is: {{ today | date:'fullDate' }}

Explanation: {{ '{{ today | date:"fullDate" }}' }} formats the date.

UpperCase Pipe Example

{{ message | uppercase }}

Explanation: {{ '{{ message | uppercase }}' }} transforms to upper case.

Currency Pipe Example

Product Price: {{ price | currency:'INR' }}

Explanation: {{ '{{ price | currency:"INR" }}' }} formats as Indian currency.

Custom Pipe Example

Original: {{ 'angular is awesome' }}

Transformed: {{ 'angular is awesome' | capitalizeWords }}

Explanation: Custom pipe to capitalize each word.
